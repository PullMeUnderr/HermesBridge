Hermes Bridge

Своя система общения, где Telegram работает как мост, а не как единственный дом.

Hermes уже умеет регистрировать пользователей через бота, создавать чаты, выдавать инвайты и связывать комнаты с Telegram-группами. Этот интерфейс нужен, чтобы тестировать все это как настоящий продукт, а не как набор API вызовов.
